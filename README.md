# TicketCompra
En este programa realizado en lenguaje Java a través del IDE Apache NetBeans, 
se crean tickets de la compra pidiendo al usuario una serie de datos por consola.
Primero el cliente ingresa su nombre y apellidos.
Después ingresa su fecha de nacimiento para que el programa calcule su edad en años.
Luego ingresa un producto, su precio y la cantidad. Una vez seleccionado se puede seguir ingresando
productos o poner "fin" para terminar.
Se imprime un ticket de compra en el que aparecen los datos que el cliente a seleccionado.
Si el cliente es menor de 18 años se aplica un descuento del 10%.
Si el cliente es mayor de 65 años se aplica un decuento del 15%.
Si el cumpleaños es en los próximos 7 días se aplica un descuento del 20%.
Si el cumpleaños es en los próximo 14 días se aplica un descuento de 10%.
Los descuentos son acumulables sin superar el 30% máximo de descuento.
Al final aparecerá el total neto de la compra tras descuentos si los hubiese.
